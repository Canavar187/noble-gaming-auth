NobleGamble – Mobile & Konsolenkompatibilität

▶ responsive-index.html – Startseite für Mobile & Gamepad-Nutzer
▶ css/mobile-style.css – Mobile-optimiertes CSS
▶ gamepad-ready.html – Gamepad-Eingabeerkennung im Browser
▶ Vollständig responsive: Smartphone, Tablet, Konsolenbrowser

Konsolennutzung:
- Funktioniert im Webbrowser von PS5, Xbox Series X
- Touch-Unterstützung für Switch & Steam Deck
- Optional mit Gamepad steuerbar (eingeschränkte Navigation)

Testen:
1. Starte index.html lokal oder auf Server
2. Öffne im Smartphone oder Konsolenbrowser
3. Drücke Knöpfe in gamepad-ready.html zur Testkontrolle
