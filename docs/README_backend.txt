NobleGamble – Backend Core Setup (MVP)

▶ Express.js REST API
▶ MongoDB Schema: User, Match
▶ Funktionen: Registrierung, Login, Match speichern & abrufen

Anleitung:
1. Node.js & MongoDB lokal installieren
2. Terminal: npm install express mongoose
3. Start: node server.js
4. API erreichbar unter http://localhost:3000/api

Weitere Routen einfach ergänzen: routes/user.js & routes/match.js
